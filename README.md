# ColetaFlow - Sistema de Gestão de Coletas

O ColetaFlow é uma aplicação web desenvolvida para a **Almeida Ambiental**, focada na gestão, organização e controle das rotas e agendamentos de coletas de resíduos. O sistema permite importar dados, organizar agendamentos e visualizar de forma clara as próximas coletas.

## 🚀 Funcionalidades
* **Importação de Dados:** Processamento de planilhas Excel para cadastro rápido de clientes e rotas.
* **Gestão de Agendamentos:** Visualização centralizada das datas de coleta.
* **Persistência de Dados:** Utiliza SQLite para garantir que as informações fiquem salvas localmente com segurança.
* **Interface Intuitiva:** Desenvolvido com foco na usabilidade operacional.

## 🛠 Tecnologias Utilizadas
* **Backend:** Python com [FastAPI](https://fastapi.tiangolo.com/).
* **Banco de Dados:** SQLite (com SQLAlchemy ORM).
* **Frontend:** HTML5, CSS3 (Templates).
* **Controle de Versão:** Git/GitHub.
